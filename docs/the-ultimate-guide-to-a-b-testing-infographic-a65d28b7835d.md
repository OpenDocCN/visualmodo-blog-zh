# 什么是 A/B 测试？

A/B 检验是对自变量和因变量之间关系的研究。例如，假设你很好奇网页字体的颜色是否会影响你的转化率。你可以测试这个。同时向[网站](https://visualmodo.com/blog/)访问者的不同部分展示同一网页的两个变体，并比较哪个变体推动更多的转化。给出更高转化率的那个获胜！

![](img/00148845294700d515d51cb436505093.png)

# 我如何运行测试？

你可能以前做过 A/B 测试，甚至不知道！你有没有给一个人发了两封一模一样的邮件，只改变了邮件中的一个变量？

基于您的假设创建一个变体，并针对现有版本进行 A/B 测试。计算测试持续时间，记住你每月的访问者，当前的转换率，以及转换率的预期变化。(在这里使用我们的贝叶斯计算器。)

# 那是 A/B 测试

A/B 测试从改变单个变量开始，如字体的颜色、字体的风格等。一旦你设置了测试，你所要做的就是研究每个测试的数据，看看哪一个测试表现更好:A，还是 b。例如，假设你想测试你的网站上的图像是如何被你的观众所感知的。为了进行 A/B 测试，你可以在你的网站上使用两张不同的图片，看看你的观众对哪张图片反应更好。

如果测试 B 的表现比测试 A 好，那么你知道你的假设是正确的。但是，如果测试 A，受控测试，比测试 B，变化的变量表现得更好，你就有数据可以得出结论，你的假设是错误的。
