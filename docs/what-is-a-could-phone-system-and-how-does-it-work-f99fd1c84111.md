# 统一通信

云电话系统是统一通信的完美推动者。统一通信就是将丰富的通信渠道和选项整合到一个接入点。云统一通信通过允许高校和客户使用他们首选的渠道，使他们与高校和客户的沟通更加一致和便捷。

因此，当您通过桌面软电话打电话并不得不离开一段时间时，您不必挂断电话，而是可以将呼叫转移到您的移动电话并继续通话。一些提供商还提供增强协作的价值，如即时消息、在线状态、CRM 集成等。
