# 通过生活优化和日常事务赢得时间

> 原文：<https://medium.com/visualmodo/gain-time-with-life-optimization-and-routines-755c4469af7c?source=collection_archive---------0----------------------->

人们发现，日常生活对你的身心健康有许多好处。许多人利用它们来腾出时间进行健康饮食和锻炼——这是两个重要但难以保持的习惯。此外，程序优化已被证明可以减少压力和焦虑。我们准备了这篇文章，所以你可以这样做，并获得寿命。

总体好处来自两个主要方面。首先是防止决策疲劳。研究表明，随着时间的推移，我们的大脑会变得疲劳，做出正确决定的能力也会下降。有了常规，很多决定都是为你做的，为一天的晚些时候节省你的脑力。

# 例程优化

惯例也减少你的认知负荷，或者完成一个过程所需的精神努力。通过每天重复同样的动作，你学会了如何完成日常任务，而不必想太多。这在你的头脑中释放了空间来学习新的任务或者将更多的注意力放在困难的事情上。

正如我们之前提到的，当你开始一项新任务时，习惯有助于转换心态。例如，如果你总是在工作前喝一杯咖啡，你会慢慢地将意义(该工作了)与例行公事(你的早晨咖啡)联系起来。最终，你的晨杯会让你有心情去完成事情。

因此，如果你发现自己很难集中注意力，仪式会特别有帮助。使用仪式训练你的大脑，帮助你以一种更有效率的心态对待工作——或者任何你难以集中注意力的任务。

# 惯例 x 仪式

虽然惯例和仪式看起来相似，但它们之间是有区别的。虽然很微妙，但这种差异在这些习惯如何优化你的日常生活中起着重要的作用。让我们简要回顾一下:

例行公事是你定期养成的一系列习惯，比如日常例行公事、锻炼例行公事或洗衣例行公事。随着时间的推移，这些习惯需要更少的思考和计划。通过执行日常事务，你可以通过减少需要做的决定的数量来简化你的生活。

仪式是你赋予了特殊意义的惯例。它们有助于改变你的心态，为某些情况做好准备。坐在办公桌前收集工作材料可能是你用来准备工作的一种仪式。晚上喝一杯茶可以帮助你入睡。

惯例和仪式可以相互配合，使你的日常任务系统化，并帮助你在它们之间平稳过渡。了解它们的不同将有助于你理解它们是如何运作的，并以最佳方式将它们融入你的生活。

# 用日常生活优化生活以赢得时间

惯例和仪式是高度个人化的，所以重要的是你要把自己的喜好和个性融入其中。我们提供了 3 个技巧来帮助你开始，并为你指明正确的方向。

# 晨间作息优化

在本文的前面，我们讨论了决策疲劳的概念，以及减少一天中早些时候做出的决策如何有助于为以后节省精力。早晨的例行公事是减少决策疲劳的关键。通过消除我们每天早上面临的选择。

你可能通常不会认为它们是决定，但每天醒来后，你必须决定早餐穿什么和吃什么。还有你是否应该洗澡，去哪里喝咖啡，你是否应该在工作前锻炼，以及那天你需要优先处理的事情。

虽然看起来微不足道，但这些选择需要能量。有了惯例，你会在醒来之前知道你前一天晚上把衣服摆好了，早餐会有酸奶加水果。你会知道你需要在跑步后淋浴，并准备好一天的工作清单。

有些人喜欢在日常生活中融入一些习惯来提高工作效率，比如整理床铺或者早起。其他人声称喝一杯水可以改善他们的身体健康，或者写日记可以提高他们的创造力。这些习惯中的任何一个都可能会打乱你的早晨例行公事。

# 健康常规

当试图平衡繁忙的工作日程和个人生活的需求时，很容易忘记练习自我照顾。这就是为什么锻炼和选择健康食品往往是难以保持的习惯。将它们融入你的日常生活可以帮助你确保它们不会从缝隙中溜走。

当你制定日常计划时，试着及时安排锻炼。举例来说，你可能一醒来就去跑步，或者在上班的路上去上一堂瑜伽课。甚至可以试着每天从办公室直接去[体育馆](https://visualmodo.com/theme/gym-wordpress-theme/)。

通过将这些优化添加到你的日常工作中，你将不会花太多时间去考虑你什么时候去锻炼，或者你是否有心情去锻炼。它只是你一天中的一部分，就像刷牙或吃饭一样。

至于食物，考虑为早餐、午餐、晚餐和小吃创建一个几个核心膳食计划的菜单。通过减少你的选择，每天吃同样的食物，不健康的食物会变得不那么诱人。每周同一天去杂货店购物，每天同一个时间准备食物，可以确保你总是有健康的食物。

即便如此，确保你的饮食平衡也是很重要的。密切关注你菜单上的配料，或者每隔一段时间轮换菜单上的某些菜肴，可以帮助确保你的[健康饮食](https://www.helpguide.org/articles/healthy-eating/healthy-eating.htm)惯例不会意外地引起问题，而不是解决问题。

# 给你的日常习惯增添意义

不管你有没有意识到，你可能已经有了一些日常习惯，稍加调整，就可以成为提高生产力的习惯。通过围绕这些习惯制定一些指导方针，并有意识地实践它们，你可以更容易地在一天中转换齿轮。

注意你难以集中注意力的地方是一个好的开始。我们已经讨论了把你的早晨咖啡变成开始工作的仪式的例子。也许相反，你在一天结束时很难放松，或者想在某个爱好上做得更好，比如演奏乐器或阅读。

# 结论

日常行为，如散步或准备你的环境。让它变得更舒适可以作为一种仪式，让你为这些活动做好准备。例如，如果你已经每天散步，你可以决定只在下班后散步。

另一个例程优化构建技术是合并对象，比如用于日志的特定笔，或者特殊的步行鞋。通过把这个物体与某种行为联系起来，使用它就成了一种仪式。举个例子，拿起你的日记笔就意味着是时候写了。穿上你的步行鞋会让你处于放松步行的心态中。
