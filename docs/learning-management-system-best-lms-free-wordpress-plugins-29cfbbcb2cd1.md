没有拖放式的课程生成器，就像许多其他插件一样。相反，你将使用常规的 WordPress 编辑器设置你的课程和教训，以及一些新的元框:你也可以嵌入视频和包含测验，还有一本指南帮助你管理学生的工作。除了免费的核心版本和 WooCommerce 付费课程，Automattic 还提供七种免费和高级扩展:
