# 提交网站的网页设计和 CSS 库列表

# 网站图库挺有意思的。从表面上看，它们是为了展示网络所能提供的最好的东西，比如巧妙的、创新的设计、聪明的代码，或者仅仅是顶级的网站。然而，它们还有更多。

人们可以浏览网页设计库为自己的网站寻找灵感。他们可以抄袭你设计中的元素，尽管他们显然不能窃取任何专有的东西而没有可能的后果。

人们可以浏览画廊，找到他们想访问的新网站。有些人非常热衷于网上审美，他们更喜欢把时间花在那些看起来和感觉都不错的网站上，而不是那些可能以更差的方式提供更好信息的网站。我正看着你呢，Reddit。

![](img/ee616a4fa8c7ef42be07fadd29c4d473.png)

最重要的是，人们浏览图库来寻找获奖的网站。这是一种荣誉，一种可以放在你的电子简历上的最小荣誉徽章。这就像是一个证明，一个封面广告，一个徽章。

网站可以将自己提交给画廊，以获得这一价值和更多。他们得到一些证明，他们做得很好，从他们的同事那里得到认可。

有趣的是，**许多这样的画廊也有链接。**他们在评判网站方面树立权威，并链接到最好的网站。现在，这些链接本身不会有很大的价值，因为它们与几乎所有其他网站都没有很高的相关性。画廊与营销业务、鞋店或技工无关。然而，它们是来自合法网站的合法链接，并且具有一些积极的价值。

最重要的是，网页设计师可以把自己提交给画廊，这样他们就可以把这些荣誉添加到自己的作品集里。然后，他们可以利用这一点来获得未来的网页设计合同。

我将在下面列出两种类型的画廊，但它们都非常相似。**其中一个专注于 CSS 中的优雅与格调**；CSS 画廊。另一个关注的是你的网页整体设计，以及你的网站有多有创意。这些是网页设计图库。此外，我有一个网站的一部分，提交您的网站到众多画廊一次。别忘了看看这些。

因为网页设计和 CSS 图库非常相似，所以我只使用一个类别。它们中的大部分都标有 CSS 图库，但是只要你的网站使用 CSS，网页设计和 CSS 图库都会为你工作。

# 终极图库列表

这个网站允许你创建一个设计师个人资料，许多其他网站也会这样做，尽管它允许你提交真实的图片，而不是自动给你的网站截图。提交能让你考虑他们的名单，但**这并不保证。费用是 9 美元，但是拒绝会退还你的申请费。您还可以被选为星级网站、每日网站或每月网站。代理商和设计师也可以在同一个档案下提交大量的设计，以获得更多的聚合价值。**

[工艺品目录](http://www.ukcrafts-directory.co.uk/)——我们希望突出我们在所浏览的网站中发现的信任信号，从而帮助这些网站展示它们是值得访问者信任的信誉良好的企业。

这个网站是一个广泛的提交网站，有三种不同的提交形式。

![](img/05287c68faffeb807782853be09cc5aa.png)

专业提交的内容会在 24 小时内得到审核，并会在他们的账户上发布(只有普通的 3300 名粉丝。)售价 20 美元。标准的提交价格是 6 美元，但会花费更长的时间，并且不会从他们的网站获得反向链接，或者拒绝退款保证。最后，**免费投稿需要你发推文。**

[Awwwards](http://www.awwwards.com/submit/) —这是一个备受瞩目的网页设计画廊，每年都展示最独特、最创新的网站。他们每年还出版一本“全球 365 个最佳网站”的书。在他们的网站上列表是非常有效的，不仅仅是为了 SEO，也是为了个人认可。如果你通过了他们严格的标准，他们 50 美元的提交费会把你加到他们的名单上，而他们 165 美元的年费会把你加到一个会员专用的目录上，客户可以在那里招募网页设计师。

[最佳 CSS 奖](http://www.bestcssaward.com/portfolio/marvel-wordpress-theme/) —最佳 CSS 奖是一个国际性的图形、应用和网站设计和开发奖项平台，旨在表彰和展示代表设计、创意和可用性的作品。每日、每月和每年的奖项都颁发给杰出的造型、创造力和技术熟练程度。获奖者将获得缎带、官方证书(根据要求)以及成千上万的最佳 CSS 奖项网站日访问量等奖励。

OnePageLove —这个独特的图库 100%致力于只有一个页面的网站。如果你的网站上有任何子页面，任何内部链接，或者除了根域主页之外的任何东西，你就没有资格。**很少有人有资格进入这个画廊**，但如果你去了，那就值得了。提交费用是 9 美元，但是如果你把你的设计作为模板出售，你可以把它提交给他们的销售清单，费用是 35 美元。

![](img/bb75e102976c44307050b63092203881.png)

[sii imple](http://siiimple.com/submit/)——自 2008 年以来一直活跃的画廊，基本上由一个人经营，依靠捐款和付费广告蓬勃发展。他们接受的网站在设计上是简单和极简的。如果你有一个有很多活动部件的复杂设计，你不符合他们的**优雅要求**，因此不会被接受。然而，提交是免费的。

[PatternTap](http://zurb.com/patterntap) —这个图库不是用来展示整个网站的。相反，它让你截屏你的网站的一部分，一个元素或 UI 的一部分。它的设计更多是为了启发其他设计师，而不是为了整个网站库共享。你必须注册才能提交，但这是免费的。

![](img/ed9171c6de41edf28c23ba13d8ee5228.png)

WeLoveWP——一个专注的图库，只接受完全基于 WordPress 作为核心内容管理系统的站点的提交。他们有四层提交。**免费让你加入他们的图库**，但不允许商业主题。Instant 的价格是 19 美元，他们的设计团队会给它一个评价。主题是 29 美元，并允许您提交商业销售主题作为广告。溢价是 69 美元，获得优先审查，并通过画廊列表提交给画廊。关于它们的更多信息，请参见下一节。

[购物车狂潮](http://cartfrenzy.com/submit-a-site/) —这个画廊只面向商业网站，特别是那些销售商品和有购物车的网站。他们有三种提交方法。其中一个是通过 Gallery Rush，下面列出的另一个提交网站，也是他们自己的。**一个是自由投稿。最后一个是快速评论，在 24 小时内浏览你的网站，价格为 3 美元，是名单上最便宜的非免费画廊之一。**

这是另一个画廊 Rush gallery，这是一个完全专注于排版网站的画廊。如果你在你的网站上对文本的设计和布局做了任何有趣的事情，甚至是实验性的设计，你可以在这里找到一个家。拉什画廊网络上还有几个网站。Minimal Exhibit 是一个极简设计网站，很像 Siiimple。博客设计英雄专注于特定博客的设计，不包括商业化的网站。最后，Folio Focus 专注于组合网站设计，通常是为摄影师设计的，尽管任何有设计组合的人都可以胜任。

![](img/f09dffee6330069e6effc3ae4e31aeb0.png)

[Pixel Perfect 作品集](http://www.pixelperfectportfolios.com/submit/)——另一个作品集画廊，这个画廊完全专注于那些用动画方法制作的作品。最初这意味着只是 Flash，但最近的一些提交更侧重于 HTML5，因为它的工作方式几乎相同。

[SaaSCSS](http://saascss.com/submit) —这是一个专门为提供软件即服务的网站设计的网页设计库。所有你的博客、电子商务网站和传统服务提供商最好走开，这里没有你想要的东西。如果你符合要求，提交是免费的。

[BuiltWithBootstrap](http://builtwithbootstrap.com/) —另一个利基提交库，这个库专注于使用 Bootstrap 构建的网站，Bootstrap 是一个更受欢迎的开源响应框架，用于移动和响应设计。提交费用是 5 美元，而且以一种独特的方式，他们通过 Fiverr 接受付款。特色提交给你主要的广告空间，但花费你 29 美元。

[不匹配的样式](http://unmatchedstyle.com/submit) —有几个部分的画廊。其中一个是时尚网站的纯画廊。一个是播客及其相关网站，这是一个有趣的利基市场，你可能没有想过。一个是策划的博客，而不是真正的画廊网站，最后一个实际上是一个编码学校，而不是画廊。

![](img/f0dd66eb7a94824b473250f2a65ab9fc.png)

这个画廊就像它们来的时候一样普通。他们没有任何限制，但他们允许社区评级。他们将提交的作品分为不同的类别，如技术、杂志、音乐、实验、未来、电子商务和教育。

Web Creme —这个画廊专注于非常时尚、近乎复古或家居风格的网站设计。如果你能想象一个版本的 Etsy，在那里出售的手工制品是网站设计，这就是你从这个画廊得到的。

[HTML5 图库](http://html5gallery.com/submit-a-site/)——这个名字恰如其分的图库要求你有一个以 HTML5 为基础的网站。这意味着您需要使用 HTML5 Doctype，您需要使用该类型的元素、属性、API 和富媒体，您需要在整个网站上使用它们，您需要演示适当的使用，而不是快速转换。没有表格，没有 WordPress 默认主题，没有临时网站是允许的。他们也有大量的积压，所以预计大约三个月的等待。

![](img/b66a0b05725edebc218794fc54a5ead6.png)

[page crush](http://www.pagecrush.com/submit/)——这个网站有一个相当大的画廊和一个**两周的等待审查的免费提交**，它给你提供了观察列表、个人投资组合列表和一条推文。不过，这只对学生和自由职业者开放，不对中介开放。代理机构必须使用 25 美元的费用选项，尽管你也需要等待大约一周。对于优先一天审查，35 美元的费用对每个人开放，并有额外的功能。

[色彩分类](http://www.colorgorize.com/) —该画廊专注于独特和创新的色彩运用，以创造愉快的体验。他们甚至有一个过滤器，所以网站的浏览器可以改变网站的色调和饱和度。

# 画廊大量提交

[CSS gallery list](http://thecssgallerylist.com/)—这个网站有一百多个不同的 CSS 图库，其中许多都列在 CSS 图库部分。它们显示了画廊的 PageRank，尽管 PageRank 本身已经过时了。提交的付款是 49 美元，由贝宝支付。

[CSS gallery submission 24x 7](http://www.cssgallerysubmission24x7.com/)—这个提交站点不仅仅是 CSS 图库。他们有 100 个 CSS 画廊包，125 个，250 个和 500 个社会书签链接包，相同数量的网站目录，提交给 80+搜索引擎，提交给 DMOZ。CSS 画廊提交是 25 美元。

这里需要注意的一点是，无论网站的声明是什么，你都不应该为“搜索引擎提交”付费。只有两个搜索引擎是重要的，那就是谷歌和必应。它们都有非常健壮的索引机制，并且会在你觉得有必要向它们提交之前很久就找到你。付费提交给任何搜索引擎，更不用说大量的搜索引擎，是不必要的，也是浪费金钱。

这个提交网站有一个严格的禁止垃圾邮件的规则，他们提交列表上的每个网站都选择了接受他们的提交。这就是为什么他们的列表比其他一些提交网站要少的原因。然而，他们确实有一个更大的图库列表，如果你愿意，你可以手动选择，上面也列出了很多。他们的提交费是 25 美元。

[MeeCSS](http://www.meecss.com/index.php) —这个提交服务有两个选项；**一个免费版本**，你可以填写基本数据，将你的网站提交给一系列精选的图库；还有一个付费版本，他们会为你提交网站，大概是利用他们的经验来优化你的提交。服务版 18 美元。有趣的是，他们通过强迫你手动填写每个图库提交的信息来鼓励你支付 18 美元，给你一个“159 步中的第 X 步”字段集。
