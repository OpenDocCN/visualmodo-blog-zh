# 更改 WordPress 帖子的日期

> 原文：<https://medium.com/visualmodo/change-date-of-wordpress-posts-bf8eb538c1f3?source=collection_archive---------0----------------------->

更改 WordPress 帖子的日期已经成为这些天 WordPress 使用的重要因素之一。无论您是对您的一篇 [WordPress](https://visualmodo.com/what-you-need-to-know-before-building-a-wordpress-website/) 帖子进行了重要编辑，并希望它以新的形式出现，还是只是想给您的一些博客帖子添加一个新的外观，您都必须更改帖子的发布日期。WordPress 允许你更改你写的任何帖子的日期。

![](img/1865e6eedead41b54514fc0a7fbe2396.png)

这个简单而安全的过程不会改变你文章的内容格式。所以，让你很容易很顺利地完成日期变更。以下是您可以承诺的两个过程。因此，没有任何耽搁，让我们看看他们:

# 谷登堡

在右侧栏中，选择“文档”，然后简单地单击日期链接，并使用古腾堡更改帖子的日期。

# WordPress 经典编辑器

您必须做的是:

*   登录您的 WordPress 仪表板。
*   转到帖子->所有帖子。
*   点击您想更改日期的帖子。
*   在右侧的“发布”框中，在“零件上已发布”处，单击“编辑”链接。

# 更改 WordPress 帖子日期的替代方法

您必须做的是:

*   登录您的 WordPress 仪表板。
*   转到帖子->所有帖子。
*   将鼠标光标悬停在帖子上，然后单击下方的快速编辑链接。
*   更改日期。
*   按下右侧的更新按钮。
