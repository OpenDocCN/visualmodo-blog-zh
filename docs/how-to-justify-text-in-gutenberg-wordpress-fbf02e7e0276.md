# 为什么要在 WordPress 中对齐文本？

网页上的段落文本对齐对网页的可读性和多样性起着至关重要的作用。“对齐”增加了单词之间的间距，并在左右边缘整齐地对齐文本。这种整洁为您的网站提供了专业的外观。通常，有四种类型的段落文本对齐方式:左对齐、正确对齐、居中对齐和对齐。
