# 选择正确的格式

没有一种简历格式能完美地适用于每一位专业人士。根据自己的技能和经验，需要选择适合自己的简历格式。

正确的简历格式将极大地提高你被招聘人员注意到的机会，并让你自己获得面试机会。以下是专业人士使用的三种标准简历格式。

*   **倒序:**这种简历格式，你的职业经历信息是按倒序写的。这以渐进的方式突出了你的技能和经验。如果你有很多没有职业空白的经验，这种简历格式是理想的。
*   **功能格式**:这种简历格式注重应聘者的技能，而不是[多年的经验](https://visualmodo.com/qualities-of-a-professional-seo-agency-you-should-look-for/)。在一个名为“技能总结”的独立部分，候选人添加了所有与他们在某些领域的专长相关的信息。此外，这种形式非常适合有间隔年的候选人和转行的候选人
*   组合简历格式:这种简历格式借鉴了倒序简历和功能简历的特点。这种形式允许候选人强调技能以及多年的经验。理想情况下，这种简历格式适合具有 10 年以上经验的高度专业化的高级专业人士。

# 创建一份专业简历摘要

招聘人员在简历上平均花费 7.4 秒。由于简历摘要是招聘人员阅读简历的第一件事，它成为决定你被列入候选名单的机会的重要部分。

以下是一些写专业应用开发者简历摘要的技巧:

*   首先，一定要写一份简历摘要，附上你多年的工作经验或证书，然后是你的职位。
*   其次，不要写很长的总结。因此，用 3-4 行来完成简历摘要
*   不要用第一人称和第二人称代词写简历摘要
*   用“拥有……方面的技能”、“非常擅长……”等等来开始简历概述。
*   最后，用段落格式写简历总结。不要使用子弹。
