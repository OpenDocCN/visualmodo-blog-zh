如何在WordPress中使用Gravatar

原文：https://medium.com/visualmodo/how-to-use-gravatar-in-wordpress-7f8c07320604?source=collection_archive---------0-----------------------#2019-12-03

给你的WordPress个人资料添加一张个人资料图片是非常容易的，你需要做的就是设置你的Gravatar账户，这是绝对免费的。在本指南中，我们将分享Gravatar WordPress的安装过程。

头像是在虚拟世界中代表你的图标或图像。事实上，它们可以让你在数百万参与各种网站和论坛的人中找到自己。好吧，你以前听说过化身。但是你听说过Gravatar吗？也许不是。这就是为什么今天我们要看看Gravatar到底是什么，以及如何在你的WordPress网站上设置一个。

在WordPress使用Gravatar的原因

Gravatar不仅仅是WordPress的一个头像系统。它也在许多其他平台上使用，如 GitHub 、Disqus(另一个评论系统)、Hootsuite、Techdirt、Mesmira和许多其他平台。当你在别人的博客上发表评论时，拥有你的个人形象，会让人们在社交媒体上看到你的个人资料或在其他博客上看到你时记住你。图像更容易记忆。所以这会让你的观众更容易记住你。

对于我们这些在各种平台上阅读、关注和评论几个不同博客的人来说，能够一次为所有人上传一张图片确实是一个巨大的便利。如果你想换头像照片，在一个地方就可以，在你发布评论的所有站点上都会瞬间改变。

因此，人们不仅会记住你和你的品牌，而且因为你在评论中有一张图片，人们会点击它并访问你的网站，特别是当他们在短时间内在多个地方看到你时。这意味着更多的交通几乎没有你的努力。只是一定不要太频繁地改变你的形象。长时间保持同一个形象，更容易让人记住你的脸，记住你的品牌。当然，确保你选择了一个好的高质量的图片，也许和你社交媒体上的图片一样，只是为了让你的品牌更加一致。记住所有这些，让我们深入一点，看看如何建立你的Gravatar档案。

Gravatar WordPress安装教程

另一个好消息是:设置Gravatar非常简单。首先进入Gravatar主页，点击“创建您自己的Gravatar”按钮。如果您已经有一个帐户，那么直接登录即可。你的WordPress账户信息将与你的Gravatar登录信息相同。如果您没有WordPress.com帐户，现在就创建您的帐户。只需输入您的电子邮件地址、用户名和密码，然后点击“继续”。然后，您会在电子邮件中收到一个确认链接。只需点击“激活帐户”按钮开始使用Gravatar。激活您的WordPress.com帐户后，点击“登录Gravatar”按钮。

因为您还没有Gravatar图像，所以登录后会要求您添加您的图像。您可以从您的计算机、网络摄像头、过去的上传或使用Web URL上传图像。出于本教程的考虑，我将向您展示如何从您的计算机上传图像。点击“上传新文件”后，它会要求您从您的计算机中选择一个文件。然后裁剪图像。确保它在小预览中看起来不错，因为大多数地方都会以那个尺寸显示。

使用限制

之后，你需要为你的Gravatar选择一个等级，以决定是否在所有网站上显示它。如果你是博客写手，你应该选择“G级”(很明显)，在所有网站上展示。就是这样。现在，你已经建立了你的第一个头像。还有一件事你需要设置，然后你就都完成了。你需要建立你的Gravatar档案。这只是一些简单的信息，当人们点击你的图片时，他们会看到你的个人资料信息。

这是一个很好的方式让人们找到更多关于你的信息，这可能会让他们以后登陆你的网站。最后，当您输入信息时，点击“保存个人资料”。现在你终于创建好了你的头像，并可以使用了。您可以设置多个具有不同电子邮件地址的头像，在不同的网站上显示您喜欢的头像。虽然我不建议这么做。保持简单，对所有网站使用一个Gravatar。

WordPress使用指南中的Gravatar

大多数人创建Gravatar账户是为了在WordPress评论上展示他们的形象。当你是一个评论者的时候，没有更多的努力。你所要做的就是，使用你在创建Gravatar帐户时使用的同一电子邮件地址。你的照片会在评论旁边。但是当你是一个WordPress网站/博客的所有者时，有更多的选择来管理头像。

您可以选择是否显示它们。默认情况下，它们出现在大多数主题中。如果没有，你可以导航到WordPress仪表盘>设置>讨论，滚动到头像部分。你也可以为Gravatars选择maxim分级许可。你还记得创建Gravatar时选择的“电影风格”评级，对吗？在管理方面也是一样的。默认情况下，G是，这意味着它可以显示给普通观众，这是我推荐你选择的。

您可以选择其他分级，但请记住分级“R”和“X”可能包含冒犯性的图像，可能会吓跑您的读者。您还可以选择默认头像的设置，以选择当留下评论的用户的电子邮件中没有Gravatar设置时出现的头像类型。默认图像有几个选项:

图像显示选项

  1. 首先，一个神秘的人:灰色的形象，可以出现在任何性别。你可能在脸书见过这个。
  2. 空白:不会出现头像。
  3. Gravatar标志:这将是公司的标志，倒置的G标志。
  4. 身份和瓦塔:这是两个会自动出现的选项。
  5. 怪物ID:另一个具有怪物形状的卡通形象，由WordPress自动创建，用户无法控制。
  6. 最后，复古:一个具有“复古”风格的低分辨率像素化图像，由WordPress创建，同样用户无法控制它。

Gravatar评级系统WordPress设置

我更喜欢使用神秘人或保持空白。永远不要用Identicaon，Wavatar，MonsterID，Retro。这些看起来垃圾邮件。如果你想添加你自己的图片，还有另外一个选择。你需要从WordPress目录安装一个名为添加新默认头像的插件。使用这个，您可以上传您想要向没有Gravatar帐户的用户显示的特定图像。请记住，添加插件只适用于WordPress.org，自托管版本。如果你使用的是免费的WordPress.com版本，那么你就不能安装它。

这提高了你的网站的品牌，但是当你开始一个新的博客并且还在弄清楚博客的所有不同元素时，这不是绝对必要的。一旦你有了一个既定的品牌，你就可以担心像这样的一个小细节。

你也可以在不添加插件的情况下做到这一点，但这意味着你需要有一些关于编码的知识，这样你就不会破坏任何代码。你可以阅读这个教程来帮助你添加自己的默认Gravatar，不需要任何插件。这可以帮助你减少网站的加载时间。
